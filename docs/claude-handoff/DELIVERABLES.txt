================================================================================
DAILY WORKFLOW (EVERY SESSION)
================================================================================

START OF EACH DAY:
─────────────────
1. Print CRITICAL_RULES_CHECKLIST.md (if not already printed)
2. Read it (5 minutes)
3. Open terminal: npm run typecheck
4. Open terminal: npm run build
5. Open browser: PORT=5199 npm run dev

BEFORE EACH COMMIT:
───────────────────
1. Run all gates: npm run typecheck && npm run build
2. Test in browser: PORT=5199 npm run dev
3. Check QUALITY_GATES.md regression test checklist (8 items)
4. Check CRITICAL_RULES_CHECKLIST.md pre-commit checklist (10 items)
5. Read your own code (code review self-check)
6. Commit with message per FEATURE_TEMPLATE.md format

WHEN STUCK:
───────────
1. Run: npm run typecheck
2. Read: TROUBLESHOOTING.md (find your error)
3. OR Read: LESSONS_LEARNED.md (find violated rule)
4. OR Read: ARCHITECTURE.md (understand data flow)
5. Apply fix, re-run gates, commit
